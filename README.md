

## Overview

This is our project that is realted to the React + TypeScript version of Lab 1.1.  
It is part of the COMP-4002 course and demonstrates building a 
simple employee directory web page using React components, TypeScript interfaces
 and Vite as the build tool.

The goal of this lab is to practice:

- Breaking down a web page into reusable components 
- Using TypeScript for type safety   
- And  Git branching `fs_lab-1.2` .



## Features

- Header component with Pixell River Logo and page title  
- Employee directory organized by department (Finance, Marketing, etc.)  
- Footer component with copyright
